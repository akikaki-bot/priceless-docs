# 短縮URLをブロックする
ここでは短縮URL荒らしに対しての対策をご紹介します。

## ブロック設定を有効する
コマンドで
```
.turlblock
```
を実行して有効にできます。
## 無効にする
有効にする方法をもう一度実行すれば無効になります。

## ブロックする短縮URLサービスを設定する。
コマンドで
```
.turlset
```

を実行して設定ができます。

## 登録済みの短縮URLサービス

|bit.ly|goo.gl|ow.ly|is.gd|Bit.do|
|:--|--:|:--:|:--|--:|

## ⚠️注意点
・この設定はサーバー全体に適応されます。

・管理者権限が必須です。
