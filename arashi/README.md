 ### コマンド一覧
 
 ※一部 `.se` コマンドに統合済み
 
 `.mute` 
 > 対象ユーザーをミュートします。
 
 ・必須権限
 `
 ロールの管理、埋め込みリンクの送信
 `
 
 `.unmute`
 >対象ユーザーのミュートを解除します。

・必須権限
`ロールの管理、埋め込みリンクの送信`

`.slowmode <Sec>`
>低速モードを設定します。

・必須権限`チャンネルの管理`
